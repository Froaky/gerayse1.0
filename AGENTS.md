# Agent Rules

This repository keeps a shared handoff file at `context.md`.

Rules for every AI agent working here:

1. Read `context.md` before making substantial changes.
2. Update `context.md` during the task, not only at the end.
3. Record high-signal information only:
   - current product scope
   - important domain rules
   - files touched
   - decisions made
   - known bugs, risks, and workarounds
   - tests run and their results
4. Prefer short factual notes over long narratives.
5. If you change behavior, add the reason and user-facing impact to `context.md`.
6. If you stop with work in progress, leave clear next steps in `context.md`.

`context.md` is the default source of project memory for future AI sessions. Keep it current.
